## Release Versioning

Before releasing, use the provided script to update version numbers automatically:
chmod +x release.sh
./release.sh patch   # bumps patch: 1.0.0+1 -> 1.0.1+2
./release.sh minor   # bumps minor: 1.0.0+1 -> 1.1.0+2
./release.sh major   # bumps major: 1.0.0+1 -> 2.0.0+2
